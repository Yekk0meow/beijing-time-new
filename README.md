# Beijing Time Sync

A simple Node.js server that displays Beijing time (UTC+8) with a static cache in the HTML for AI to read.

## Deployment

1. Deploy to **Railway**: Import this GitHub repository.
2. Deploy to **Vercel**: Import with default settings.

## Features

- Real-time Beijing time display
- Static cache in HTML for AI access
- Simple and lightweight

## Files

- `server.js` - Main server
- `package.json` - Node.js configuration
- `.gitignore` - Ignores node_modules
